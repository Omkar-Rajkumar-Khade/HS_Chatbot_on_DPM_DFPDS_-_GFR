# Chatbot on DPM, DFPDS & GFR


* Elastic Search Database Setup

##### To start VectorDB container
###### For Linux
```bash
sudo docker-compose up -d
```

###### For Windows
```bash
docker-compose up -d
```

##### To Stop VectorDB container
###### For Linux
```bash
sudo docker-compose down
```

###### For Windows
```bash
docker-compose down
```
